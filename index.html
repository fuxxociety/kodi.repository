FuxXocietY Kodi Repository [https://fuxxociety.github.io/kodi.repository/]

Enable Installation of add-ons from unknown sources:

  • Go to “System Settings” > “System” > “Add-ons”
  • Enable “Unknown sources”
  • Click “Yes”

Steps to add source:

  • Go to “System Settings” > “File Manager” > “Add Source”
  • Click “<None>”
  • Type “https://fuxxociety.github.io/kodi.repository/”
  • Enter “FuxXocietY” as name for this media source
  • Click “OK”

Steps to add repository:

  • Go to “System Settings” > “Addons” > “Install from zip file” > “FuxXocietY”
  • **note** Kodi will only display folders, and files that begin with 'repository' ?
  • Install repository.fuxxociety-0.0.1.zip
  • Complete Add-on repository installation

Steps to install add-on:

  • Go to “System Settings” > “Addons” > “Install from repository”
  • Click “FuxXocietY Kodi Repository” > “Video add-ons”
  • Click “FuxXocietY”
    • Use “FuxXocietY for Kodi 18.x Leia (xbmc.python 2.x)
    • Use “FuxXocietY for Kodi 19.x Matrix (xbmc.python 3.x)
  • Click “Install”
  • Click “OK”
